Market Game

This project was assigned for completion in my AP Computer Science class where anything was to be made with HTML / CSS, and JavaScript. The project I decided to create was a fast-pace market simulation game. The game consists of a light item market containing minimally expensive items and a large item market containing expensive items.
The purpose of this program was to simulate a fast pace market environment that rewards quickly buying at a low price and selling high where the product is still competitively priced. The market game at its most fundamental is a large market that sells expensive items with a longer refresh time and a light market that sells less expensive items at a faster refresh interval. The items can be bought from either the light market or the large market if there are sufficient funds. When an item or group of items is purchased the item is stored in the user's inventory. From the inventory, the user can select the quantity of each item type to sell and the price in which the items will be sold. The motive of the game is not indifferent from an actual market as items can be bought for a low price and sold for a high price. However, the better the deal the item is the faster it will sell in the live market. If the item sells it will be shown as crossed out and it will no longer be available for purchase. A similar time related selling algorithm is implemented on the users' own market. This algorithm is an exponential function where x is the price and y is the amount of time it will take for the item to sell. The game starts at $1000 and from there it is up to the user to make money by buying both the light and large items at a low price and selling high.

Focused Bird Watcher

This is a game based on taking clear pictures of birds. Based on how clear the photo is and how rare the bird is, the program will pay more for a picture. The user starts the game with the three common birds and must make money to learn what other, rarer, birds look like. Once this is learned, the user may find these birds in the wild and as they are rarer, pictures of them can be sold for more. However, these birds are not only less common to spawn, but they are quicker to fly away. This game is built using bootstrap and VanillaJS. The user's money is saved as well as what birds they have learned.

Fast Reader

This project takes pasted text content and displays it one word at a time in the center of the screen. The words per minute is chosen by the user and the time that the content will take to read is displayed. The idea of the project is reading may be able to be made faster if the reader's eyes never had to move. If the computer displays the text in a way such that all that is required is to watch and understand the text, then it may be possible that the reading process could be sped up. In actuality, does it work? Well, maybe. Try it yourself, see how fast you can read!

Braille Converter

This braille website is a place to learn about how English characters are represented in braille. The website also serves as a method in which to test oneself; English letters can be randomly generated and the user can input the corresponding raised dots to test for correctness. If the correct braille symbol is made and submitted, the time will be recorded and stored in the browser's local memory. If the incorrect braille symbol is created, however, the correct braille combination will be shown below where the braille character was submitted.
The website also includes a feature in which English sentences comprised of letters can be translated into braille sentences.
